# 轻期

微信生态内极简生理期记录小程序。数据纯本地化，无需登录，专注经期记录、周期预测与系统日历提醒。

## 功能概览

### 首页

- 月视图日历，支持左右滑动切换月份
- 顶部状态卡片：展示当前经期状态或距下次经期天数
- 日历标记：已记录经期、预测经期、排卵日、易孕期
- 点击日期查看当日详情，支持快捷记一笔
- Smart Banner：首次记录后提示去设置真实周期
- 状态卡片 ℹ️ 图标可唤起「预测原理」半屏弹窗

### 记录

- 经期历史列表，展示开始/结束日期与持续天数
- 半屏弹窗记一笔：「大姨妈来了」/「大姨妈走了」
- 支持删除单条记录
- **数据导入 / 导出**：JSON 格式备份与恢复（记录页顶部工具栏）

### 统计

- 近 6 个月周期长度、经期长度折线图
- 连续记录 3 个月以上自动启用加权平均算法

### 我的

- **周期与预测设置**：经期长度（2–14 天）、周期间隔（20–45 天）
- **提醒设置**：经期来临提醒、排卵日提醒（写入系统日历）
- **我们的预测原理**：算法说明与免责声明
- **隐私政策**：本地数据说明 + 微信平台隐私指引入口
- **模式切换**：日常模式 / 备孕模式（高亮排卵日与易孕期）
- **主题配色**：玫瑰粉、薄荷绿、莫兰迪灰、克莱因蓝
- **清除所有数据**：一键删除本地全部记录

## 系统日历提醒

提醒通过 `wx.addPhoneCalendar` 写入手机系统日历，无需服务端、无需联网。

| 类型 | 触发时机 | 提醒时间 | 默认 |
|------|----------|----------|------|
| 经期来临 | 每次记录「经期开始」后，按最新预测自动添加 | 预计经期前 1–2 天的 18:00 | 开启，提前 2 天 |
| 排卵日 | 同上 | 预计排卵日当天 18:00 | 关闭 |

**机制说明：**

- 单次非全天事件，`alarmOffset: 0`（到点提醒）
- 自动去重，同一预测日期不重复添加
- 提醒日已过期则跳过
- 须在用户点击手势链中同步调用 API
- 预测变化后，旧日程需在系统日历 App 中手动删除

**权限与合规：**

- 需用户授权「添加到日历」（`scope.addPhoneCalendar`）
- 正式版须在 [微信公众平台](https://mp.weixin.qq.com/) 完善「用户隐私保护指引」，声明 **日历（仅写入）权限**（`CalendarWriteOnly`），审核通过并重新发布后生效
- 已接入微信隐私授权弹窗（`privacy-popup` 组件 + `wx.onNeedPrivacyAuthorization`）

## 技术规范

- 微信小程序原生框架（WXML + WXSS + JS）
- `lazyCodeLoading: "requiredComponents"` 按需注入
- `style: "v2"` 新版样式规范
- WebView 渲染，兼容 Android / iOS
- `rpx` 响应式 + `env(safe-area-inset-bottom)` 适配刘海屏
- 自定义 TabBar
- 数据纯本地化（`wx.setStorageSync`），不上传云端
- 日历 API 要求基础库 2.15.0+

## 本地开发

1. 安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 用开发者工具打开本项目目录
3. 在 `project.config.json` 中配置 AppID（或使用测试号）
4. 编译预览；日历提醒需在真机测试

## 核心算法

- 默认经期 5 天、周期 28 天
- 排卵日 = 下次预测经期开始日 − 14 天
- 易孕期 = 排卵日前 5 天 + 后 4 天
- 连续记录 3 个月以上启用加权平均法
- 超过 45 天未记录则暂停预测

## 隐私说明

- 经期记录、周期设置等核心数据仅保存在用户设备本地
- 不上传云端，不用于广告分析
- 首次启动需同意应用内隐私政策
- 使用日历提醒时，需额外同意微信平台《用户隐私保护指引》

## 正式发布 checklist

上线日历提醒功能前，请确认：

- [ ] 公众平台已声明「日历（仅写入）权限」并审核通过
- [ ] 提审版本未勾选「本小程序未采集用户隐私」
- [ ] 隐私指引与代码实际调用的接口一致
- [ ] 审核通过并发布线上版后，在真机复测添加日历

常见错误码：

| errno | 含义 | 处理 |
|-------|------|------|
| 1025 | 隐私接口被封禁 | 完善后台隐私指引，审核通过后自动解封 |
| 112 | 接口未在隐私协议中声明 | 后台补充日历隐私类型声明 |
| 101102 | 用户未同意隐私协议 | 引导用户同意微信隐私弹窗 |
